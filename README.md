# Chrome-quick-Furigana-search-extension

Instead of **selecting kanji** $\rightarrow$ **ctrl c** $\rightarrow$ **new tab** $\rightarrow$ **paste and add "in hiragana"** (*4 steps*). 

You can now **selecting kanji** $\rightarrow$ **right click** and there will be a **quick search option** for you (*2 steps*).

You can alternatively choose between **Mazii** and **Jisho** for different kanji look up.

### How to use:
1. **Download** the folder.
2. Open **Chrome** and go to **extensions** $\rightarrow$ **manage extensions**.
3. Click **load unpacked**.
4. **Select the folder**
